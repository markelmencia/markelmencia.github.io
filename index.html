I'm Markel Mencía

A Computer Engineering student interested in cybersecurity and operative systems at the University of Deusto.

Good morning! I'm Markel. I've always enjoyed computers and difficult stuff. That's why I got into cybersecurity. When I'm not playing games or the piano in my free time, I like to build my own projects about different topics. I also like banging my head against CTF challenges, often under the name solstice/slstc, for my university team: 0xDECODE.

Here's some of the things I've done!

  • rvik: RV32i Assembler and Simulator

    • rvik is an open-source assembler and simulator built for fun in Java. It is able to assemble the RV32i instruction set variant of the open-source ISA RISC-V. It also provides an interface that allows you to simulate the execution of RV32i programs. You can learn more about it in its GitHub repository.

  • LITE-16: A 16-bit Instruction Set Architecture

    • LITE-16 is a basic ISA I designed after my Computer Structure course, intended to be as straight-forward as possible to show the basics of the Assembly Language and how CPUs work. You can read its documentation and logic architecture in its repository.
