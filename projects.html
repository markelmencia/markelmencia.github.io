My Projects

This is some of the personal work I've done over time.

gochat

See more

gochat is a light, TCP-based Client-Server chat application that runs its own application protocol. Programmed in Golang, this is the project I've spent the most time on. I made it with my friend Sprinter, and it took dozens of hours of designing and coding.

rvik

See more

rvik is my own RV32i variant simulator and assembler implementation. This project includes a text editor programmed in Java that is able to assemble RV32i instructions. The project has had multiple version and refactors, and it's been a key part in my learning of OOP development.

LITE-16

See more

LITE-16 is a 16-bit Instruction Set Architecture designed entirely by me in Verilog. It is inspired in the RISC-V ISA, and it aims to be as straight-forward as possible. The project also includes a LITE-16 assembler programmed in C.

0xDECODE: GNU-Linux

See more

This project is a free, open-source course on the fundamentals of the GNU/Linux operative system. It was done as a resource for 0xDECODE, the university group I'm involved in. The course contains a full book written by me in spanish, and a set of exercises to put in practice the content of the book.

Current project

Lately I've been interested in parallel environments and in the way in which they help improve the performance of certain computational activities. Since I've always had an interest in compression algorithms, this is a perfect excuse to work with them.


One my current projects is to investigate about the performance of parallel computing (with tools like CUDA) on compression algorithms. There's already published work about this topic, but it is an area I'd like to investigate by myself. Being a complete beginner on both compression algorithms and parallel computing, I'm sure it'll be no easy task, but I'm making some progress on it.
